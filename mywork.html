New Orleans Quiz

  • Live App GitHub Repo

    • This app quizzes the user on certain historical and geographical elements about New Orleans.

    It gives immediate feedback upon the user's selection letting them know whether the answer they selected was correct or incorrect.

    If it was incorrect, the correct answer is provided in the feedback.

    At the end of the quiz, the user can choose to start the quiz over.

    The app utilizes html, css, javascript, and jQuery.
